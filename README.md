# Upload M3U8 - PandaVideo

Este projeto permite fazer upload de vídeos M3U8 para a plataforma PandaVideo via API.

## Instalação

1. Clone o repositório
2. Execute `npm install` para instalar as dependências

## Uso

1. Execute `npm start` para iniciar o servidor
2. Abra o navegador em `http://localhost:3000`
3. Preencha o formulário com a URL M3U8, título, chave da API e outros campos opcionais
4. Clique em "Fazer Upload" para iniciar o processo de upload

## Requisitos da API

- Chave da API válida do PandaVideo
- URL M3U8 acessível

## Desenvolvimento

O servidor roda na porta 3000 por padrão.
